# OS-Explorer

OS-Explorer is an interactive operating systems simulation platform designed to help users visualize and experiment with core OS concepts through step-by-step simulations and interactive modules.

The project provides a hands-on learning environment for understanding process scheduling, memory management, deadlocks, paging, synchronization, file systems, disk scheduling, interrupts, and other fundamental operating system concepts.

---

## Features

### CPU Scheduling
- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Round Robin
- Priority Scheduling

### Memory Management
- Fixed and Variable Partitioning
- Memory Allocation Techniques
- Virtual Memory Simulation

### Page Replacement Algorithms
- FIFO
- LRU
- LFU
- MRU
- Optimal Page Replacement

### Deadlocks & Resource Allocation
- Banker's Algorithm
- Resource Allocation Graph (RAG)
- Deadlock Detection & Avoidance

### Process Synchronization
- Reader-Writer Problem
- Dining Philosophers Problem
- IPC Concepts

### Disk Scheduling
- FCFS
- SSTF
- SCAN

### Additional Modules
- File Allocation & Organization
- System Calls
- Interrupt Handling
- I/O Management
- Booting Process Simulation
